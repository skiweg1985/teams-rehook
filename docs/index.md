# Documentation

Teams Rehook relays webhook messages from internal systems into Microsoft Teams through managed routes, Teams bot conversation references, and optional Microsoft Graph-backed delivery.

If you are a user, start with the [User guide](user-guide.md). If you operate the service, start with the [Admin guide](admin-guide.md) and [Configuration](configuration.md). If you develop or review the code, start with the [Developer guide](developer-guide.md) and [Architecture](architecture.md).
